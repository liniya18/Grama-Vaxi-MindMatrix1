# Grama Vaxi MindMatrix

Grama Vaxi MindMatrix is an Android application developed for managing animal vaccination records and healthcare activities in rural areas.

## Features

* Animal Registration
* Vaccination Records
* Reminder Notifications
* Kannada Language Support
* Voice Input
* PDF Report Generation
* AI Health Suggestions

## Technologies Used

* Kotlin
* Jetpack Compose
* Room Database
* MVVM Architecture

## Installation

1. Clone repository
2. Open in Android Studio
3. Run project
